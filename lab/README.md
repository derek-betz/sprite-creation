# Sprite Lab

Use `lab/` for exploratory sprite work that is not yet curated enough for `assets/` or tied tightly enough to a real app to live under `integrations/`.

Good fits:

- random character concepts
- motion tests
- prompt experiments
- roadway/status indicator ideas
- palette and silhouette studies

When something becomes reusable, move it into `assets/` and add it to `catalog/asset-manifest.json`.
